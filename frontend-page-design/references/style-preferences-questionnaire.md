# 视觉风格偏好问卷

在开始 UI 美化前，先询问用户以下问题。未确认前，不要做强风格改造。

## 必问项（至少确认）

1. 视觉气质
- 偏好：`专业简洁` / `活力营销` / `科技未来` / `内容阅读` / `潮流个性`

2. 配色方向
- 让用户从 `color-palettes.md` 中先选 1-2 套候选，再确定最终色板。

3. 动效强度
- 选项：`无动效` / `轻动效（推荐）` / `中等动效` / `强动效`
- 说明：默认推荐轻动效，仅在状态变化和关键反馈处使用。

4. 毛玻璃（Glassmorphism）
- 选项：`不要` / `少量局部` / `可作为主要风格`
- 说明：若使用毛玻璃，需确保对比度与文字可读性。

5. 阴影与层次
- 选项：`极轻阴影` / `标准阴影` / `明显层次阴影`

6. 圆角风格
- 选项：`锐利（2-4px）` / `中等（6-10px）` / `圆润（12px+）`

7. 渐变使用
- 选项：`纯色为主` / `局部渐变` / `较多渐变`

## 常见风格组合模板

## 模板 A：稳重企业

- 动效：轻动效
- 毛玻璃：不要
- 阴影：极轻阴影
- 圆角：中等
- 渐变：纯色为主

## 模板 B：现代科技

- 动效：中等动效
- 毛玻璃：少量局部
- 阴影：标准阴影
- 圆角：中等或圆润
- 渐变：局部渐变

## 模板 C：营销视觉

- 动效：中等到强动效
- 毛玻璃：可局部使用
- 阴影：明显层次阴影
- 圆角：圆润
- 渐变：较多渐变

## 模板 D：极简内容

- 动效：无动效或轻动效
- 毛玻璃：不要
- 阴影：极轻阴影
- 圆角：锐利或中等
- 渐变：纯色为主
