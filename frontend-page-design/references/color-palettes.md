# 配色方案参考

使用方法：
- 首先根据产品气质（专业、活力、极简、科技、内容型）筛选 3-5 套候选。
- 让用户从候选中选 1 套，或指定“主色沿用 A、背景沿用 B”的混搭方式。
- 最终落地时将颜色映射为 token：`--color-bg`、`--color-surface`、`--color-text-primary`、`--color-text-secondary`、`--color-primary`、`--color-accent`、`--color-border`。

## 快速确认问题

1. 你希望页面更偏「专业稳重 / 活力增长 / 科技未来 / 极简中性」哪一类？
2. 你更偏好浅色、深色，还是同时支持两套主题？
3. 是否接受高饱和强调色（更吸睛）？
4. 是否有品牌色必须保留？
